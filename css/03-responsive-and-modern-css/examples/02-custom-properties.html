Custom Properties (CSS Variables)

Colors and spacing here come from --primary-color, --bg, --fg, --spacing-unit, etc., declared once on :root and consumed via var(--name).

This page also responds to your OS's dark mode setting via @media (prefers-color-scheme: dark), which redeclares the SAME variable names with dark values — every rule using var(--bg)/var(--fg)/etc. updates automatically, with no duplicated rules.

This box has class theme-dark, which re-declares the same variables to dark values in THIS SCOPE only — demonstrating that custom properties cascade/inherit and can be overridden per subtree, independent of the OS preference.

Box using var(--undefined-var, 2px dashed gray) — the variable was never declared, so the fallback value is used.

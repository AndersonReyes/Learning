Fluid heading via clamp()

font-size: clamp(1.5rem, 4vw + 1rem, 3rem) — stays at 1.5rem on narrow viewports, scales with viewport width in the middle, and caps at 3rem on wide viewports.

width: min(100% - 32px, 600px) — never exceeds 600px, but shrinks to fit narrow viewports (minus 32px for padding).

max() for a sidebar that grows with viewport

width: max(150px, 20vw) — at least 150px, but grows past that on wide viewports.

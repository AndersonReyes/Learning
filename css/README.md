# CSS

Notes and viewable examples for CSS — presentation: layout, color,
typography, responsiveness.

## How to use this track

There's no test runner or build step. Each topic folder has:

- `notes.md` — concept explanations with inline code snippets.
- `examples/*.html` (and occasionally a paired `.css`) — standalone, complete
  pages with self-contained `<style>` blocks.

Open any `examples/*.html` file directly in a browser (double-click it, or
`open examples/01-selectors-and-cascade.html` on macOS / `xdg-open` on Linux)
to see it rendered. Read `notes.md` alongside the example it references.
Resize the browser window for the responsive examples in
[`03-responsive-and-modern-css`](./03-responsive-and-modern-css).

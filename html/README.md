# HTML

Notes and viewable examples for HTML — structure and meaning, not appearance
(CSS) or behavior (JS).

## How to use this track

There's no test runner or build step. Each topic folder has:

- `notes.md` — concept explanations with inline code snippets.
- `examples/*.html` — standalone, complete HTML documents.

Open any `examples/*.html` file directly in a browser (double-click it, or
`open examples/01-document-structure.html` on macOS / `xdg-open` on Linux)
to see it rendered. Read `notes.md` alongside the example it references.

## Topics

- [`01-html-fundamentals`](./01-html-fundamentals/notes.md) — document
  structure, text and semantic elements, semantic layout, links and images,
  lists and tables, forms.

No exercises in this track — see [`javascript/`](../javascript) for
test-driven exercises.

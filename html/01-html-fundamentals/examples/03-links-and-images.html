Links and Images

Link types

  • Absolute URL — full address including protocol and domain.
  • Root-relative URL — starts from the domain root (a leading /).
  • Relative URL — resolved relative to this file's directory.
  • Same-page anchor — jumps to the id="fig1" element below.
  • Opens in a new tab — paired with rel="noopener noreferrer" so the new page can't access window.opener.

Images

Every <img> needs an alt attribute describing the image for screen readers and for when the image fails to load. width/height let the browser reserve space before the image loads, preventing layout shift.
